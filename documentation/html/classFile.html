Detailed Description

Retrieves data about bayfiles files by sending requests to http://api.bayfiles.com/v1/account/.

Represents a Bayfiles file. fileId and infoToken are required in order to get the others attributes, except deleteToken and owner which are provided by the Account::getFiles() method.
